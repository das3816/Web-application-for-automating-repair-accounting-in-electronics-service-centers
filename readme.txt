Інструкція із запуску вебзастосунку 

Крок 1: Налаштування та запуск бази даних (MongoDB)

Оскільки проєкт використовує базу даних MongoDB, перед запуском сайту переконайтеся, що на вашому комп'ютері встановлено та запущено локальну базу даних.

1. Перевірте файл `.env` у папці `backend`. Рядок підключення має виглядати так:
env
MONGO_URI=mongodb://127.0.0.1:27017/repair_service_db


2. Переконайтеся, що програма **MongoDB Community Server** встановлена на комп'ютері (вона запускається автоматично як служба Windows при старті системи).


Крок 2: Запуск Бекенду (Сервера)

1. Відкрийте перше вікно Термінала (або Командного рядка / CMD).
2. Перейдіть до папки з бекендом за допомогою команди:
   cmd
   cd "C:\Users\DAS\Documents\SmartStorage\Veb-zastosunokdlyaavtomatizaciioblikuremontivu_dxdhxq2k.iop\Веб-застосунок для автоматизації обліку ремонтів у сервісних центрах електроніки\web-site\backend"


3. Якщо ви запускаєте проєкт уперше, встановіть необхідні бібліотеки (залежності):
cmd
npm install



4. Запустіть сервер командой:
   cmd
   npm start



5. Важливо: Не закривайте це вікно термінала. Сервер повинен працювати постійно, поки ви користуєтеся сайтом.



Крок 3: Запуск Фронтенду (Клієнтської частини)

1. Відкрийте друге (нове) вікно Термінала.
2. Перейдіть до папки з фронтендом:
cmd
cd "C:\Users\DAS\Documents\SmartStorage\Veb-zastosunokdlyaavtomatizaciioblikuremontivu_dxdhxq2k.iop\Веб-застосунок для автоматизації обліку ремонтів у сервісних центрах електроніки\web-site\frontend"

3. Якщо запускаєте вперше, встановіть залежності для фронтенду:
   ```cmd
   npm install


4. Запустіть локальний сервер розробки:
cmd
npm run dev

   *(Примітка: якщо виникне помилка, спробуйте команду `npm start`).*


Крок 4: Перехід на сайт

1. Після запуску фронтенду в другому терміналі з'явиться посилання (найчастіше це `http://localhost:5173` або `http://localhost:3000`).
2. Скопіюйте це посилання, вставте його в адресний рядок вашого браузера (Google Chrome, Edge тощо) та натисніть **Enter**.
3. Сайт готовий до роботи! Тепер ви можете реєструватися, входити в систему та тестувати весь функціонал.


Як вимкнути сайт після роботи?
Щоб зупинити роботу сайту, перейдіть по черзі в кожне з двох вікон термінала і натисніть комбінацію клавіш Ctrl + C, після чого закрийте вікна. При наступному запуску кроки з `npm install` повторювати вже не потрібно — достатньо буде просто виконати `npm start` для бекенду та `npm run dev` для фронтенду.
